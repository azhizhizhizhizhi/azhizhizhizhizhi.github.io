hexo与github搭建博客之搭建环境

网上有大量教程，我就挑一些比较常见的坑进行说明吧

关于hexo和github 搭建个人博客一些踩过的坑和一些简单搭建步骤说明吧

主要是在windows环境下的搭建

  • nodejs 服务的安装和配置
  • Git下载使用
  • hexo 的下载
  • github账号的申请和建立代码仓库

1.nodejs 安装

1.可以到官网进行下载node下载

2.检查nodejs在本地安装成功没有

ｗｉｎ＋Ｒ　输入ｃｍｄ　输入node -v 查看有没有显示版本号。显示则成功，不显示则重新尽行安装操作。

2.git下载使用

1.git 是一个版本控制工具，通常配合着 github 一起使用，进行源代码托管。如果你还没有 github 账户，先去注册一个吧。这里给个 传送门

2.下载git，安装git，安装好git之后进行配置git的环境。到桌面上进行鼠标右键点击git bash 进入命令界面之后，修改本地git全局参数

为了在后续操作中我们能将本地仓库的代码推送至 github 的仓库上，我们需要在本地生成 SSH 秘钥，并将公钥保存到 github 账户信息中，这样我们在本地提交的时候 github 就能通过本地的私钥与公钥进行校验。

生成密钥的过程中会有些提示要求输入，我们一路回车下去即可。
